### 4. Salary Calculation

- **Allowances:**
  - **HRA:**
    - 30% if Basic ≤ 1000
    - 25% if Basic ≤ 3000
    - 20% if Basic > 3000
  - **DA:** 30% of Basic
  - **Conveyance:**
    - ₹50 if Basic ≤1000
    - ₹75 if Basic ≤2000
    - ₹100 otherwise
  - **Entertainment:** NIL if Basic ≤1000; else ₹100
- **Deductions:**
  - PF → 6% of Basic
  - Group Insurance:
    - ₹40 if Basic ≤1500
    - ₹60 if Basic ≤3000
    - ₹80 otherwise
- **Salary Formulas:**
  - Gross Salary = Basic + HRA + DA + Conveyance + Entertainment
  - Total Deduction = PF + Insurance
  - Net Salary = Gross - Deduction
